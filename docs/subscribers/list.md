# Subscriber List

The subscriber list is your main view of every customer. It shows who is online right now, who is suspended, and lets you act on any account.

## Finding a Subscriber

Use the **search box** to find someone by name, email, phone number, or username.

Use the **filter dropdowns** to narrow the list by status, zone, device, or service type. Filters stack — you can combine them (e.g. Suspended subscribers in a specific zone).

## Acting on a Subscriber

Click the actions menu ( ⋮ ) on any row:

| Action | What it does |
|---|---|
| View Details | Opens the full subscriber detail page |
| Suspend | Removes internet access |
| Reactivate | Restores internet access (appears when suspended) |
| Send Message | Opens SMS/email dialog for this subscriber |
| Setup RADIUS | Configures RADIUS credentials (PPPoE subscribers) |
| Continue Draft | Resumes activation for draft accounts |

## Top Bar Buttons

| Button | What it does |
|---|---|
| Add Subscriber | Create a new subscriber account |
| Import | Upload subscribers in bulk from a CSV file |
| Export CSV | Download the current filtered list as a spreadsheet |
