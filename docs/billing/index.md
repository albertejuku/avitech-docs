# Portal Overview

CoreSync is an ISP management platform that handles billing, subscriber management, network device provisioning, and support — all from a single web portal.

## What You Can Do

| Area | What it covers |
|---|---|
| **Dashboard** | Live overview of active subscribers, revenue, device status, and alerts |
| **Subscribers** | Create and manage subscriber accounts, assign plans, monitor online/offline status, send SMS and email |
| **Billing** | Service plans, subscriptions, payments (including M-Pesa STK Push), invoices, and financial reports |
| **Network** | Routers, OLTs, NAS devices, zones, IP pools, and active PPPoE/DHCP sessions |
| **Support** | Ticketing system, automations, and customer communication |
| **Settings** | System configuration, user management, roles and permissions, notification templates |

## Portal Layout

After logging in, the portal has two main navigation areas:

- **Sidebar** — lists every section: Dashboard, Subscribers, Billing, Network, Support, and Settings. Click any section to expand its sub-pages.
- **Top bar** — search, notifications, and your user profile menu.
