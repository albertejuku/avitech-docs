# Zones

Zones represent your coverage areas — neighborhoods, buildings, regions, or any logical grouping. Every subscriber is assigned to a zone, and zones determine which network device serves the subscriber.

## Viewing Zones

Go to **Network → Zones** in the sidebar.

## Creating a Zone

1. Click **Add Zone**
2. Enter a **Zone Name** (e.g. "Kileleshwa", "Block A", "Downtown")
3. Optionally add a description
4. Click **Save**

## Assigning Subscribers to Zones

When [adding a subscriber](../subscribers/creating.md), select the zone from the dropdown. The zone determines:

- Which router or NAS device the subscriber connects through
- How the subscriber appears in filtered lists and reports

## Editing and Deleting Zones

- **Edit** — change the zone name or description. Existing subscribers in the zone are unaffected.
- **Delete** — only possible if no subscribers are assigned to the zone. Reassign subscribers to another zone first.

!!! tip
    Start with broad zones and add more as your network grows. You can always move subscribers between zones later.
