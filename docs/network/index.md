# Network Portal

The Network Portal is where you configure the physical infrastructure that delivers internet to your subscribers — speed profiles, coverage zones, network devices, and IP address pools.

## Sections

| Section | What you do there |
|---|---|
| [Speed Profiles](speed-profiles.md) | Define bandwidth tiers (10Mbps, 20Mbps, etc.) that plans reference |
| [Zones](zones.md) | Create coverage areas that group subscribers by location |
| [Devices](devices.md) | Register routers, OLTs, and NAS devices that authenticate subscribers |
| [IP Pools](ip-pools.md) | Configure IP address ranges for Static IP subscribers |

## Typical Workflow

1. Create speed profiles first — plans and devices depend on them
2. Define zones for your coverage areas
3. Register all network devices and assign them to zones
4. Configure IP pools (if offering Static IP services)
5. Create service plans in the Billing Portal — linking to your speed profiles
6. Add subscribers and assign them to zones and devices
